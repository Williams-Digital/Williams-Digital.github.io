Pattern Reference
SMPTE Color Bars (SMPTE EG 1-1990): The standard color bar test pattern used in NTSC television. The top 67% contains seven vertical bars: white (75%), yellow, cyan, green, magenta, red, and blue. The middle strip contains blue, magenta, cyan, white, and the -I/+Q signals. The bottom strip (PLUGE) contains a black set reference at 3.5, 7.5, and 11.5 IRE for setting monitor brightness.
